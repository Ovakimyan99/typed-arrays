# Typed Structures over TypedArray

Структуры данных поверх `ArrayBuffer` / `TypedArray` с явной схемой, выравниванием и ленивыми геттерами/сеттерими.


### Выигрыш в скорости TypedArray

При классическом подходе затрачивается время на парсинг JSON, аллкацию множества объектов.
Что также не бесплатно при сборке мусора и доступе к данным. Здесь же мы описываем единожды схему
и читаем/пишем напрямую в ArrayBuffer без парсинга и аллокаций.
